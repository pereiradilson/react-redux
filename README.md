# Project with React Redux

<h3 align="center">
  A simple project with Redux
</h3>

## Installation NodeJS

Installing [NodeJS](https://nodejs.org/en/download/).

## Usage

```bash
# Cloning project
git clone https://github.com/pereiradilson/react-redux.git

# Install dependencies
yarn
```

## Run fake back-end

```bash
# Executing back-end
yarn server
```

## Run front-end

```bash
# Executing front-end
yarn start
```
